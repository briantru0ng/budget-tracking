# Settings & Tools

Configuration and maintenance tools organized into six tabs.

## Baskets Tab

Manage the category hierarchy. Baskets are top-level groups (e.g., Food, Housing, Services). Sub-categories sit inside baskets and are what transactions get tagged with.

- View all baskets and their sub-categories
- Add or remove sub-categories within any basket
- Create entirely new baskets

## Apply Interest Tab

Manually compound interest on savings goals that have an APY rate set.

- Select a goal, choose how many months to compound
- Preview the interest amount and new balance before applying
- "Apply to All Goals" button compounds 1 month across every interest-bearing goal at once

## Income Breakdown Tab

Analyze income streams over a date range.

- Pie chart and table of income by source
- Teach income categorization rules (keyword to stream mapping: Salary, Side Income, Investments, Gifts, Refunds, Government, Rental, Business)

## Duplicates Tab

Scan and clean up duplicate transactions. Four detection methods:

1. **Exact duplicates** — identical Date + Description + Amount
2. **Likely duplicates** — same date, amount, and normalized merchant but different raw descriptions
3. **Suspicious matches** — same date, amounts within $0.10 of each other
4. **Multiple same-day purchases** — same merchant on the same day (usually legitimate)

"Auto-Remove Duplicates" deletes all exact duplicates, keeping the first occurrence.

## Categorization Tab

- **Quick Teach** — add keyword-to-category rules
- **Merchant Normalization** — map messy bank descriptions to clean merchant names
- **Recategorize All** — re-run all rules across every transaction (useful after adding new rules)
- **Uncategorized count** — shows how many transactions still need sorting

## Budgets Tab

Set monthly budget targets per category. Enter a dollar amount for each spending category. Budgets drive the alerts on the Live View page — any category projected to exceed its budget triggers a warning.

Current budgets are displayed as a summary table with the total monthly budget.
